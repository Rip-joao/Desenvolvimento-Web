Oque é Semântica

😁Semântica é sobre dar sentido verdadeiro às coisas. No HTML5, isso significa usar elementos que descrevem claramente o conteúdo que carregam.
Por exemplo, em vez de usar pra tudo, podemos usar (header) para cabeçalhos, (nav) para menus de navegação, (footer) para rodapés...
Cada tag tem um significado específico, como se estivesse contando uma história sobre o que está dentro dela 📖.
Isso é importante porque:
✨ Ajuda os navegadores e motores de busca a entenderem melhor sua página, como se ela tivesse legendas.
🦻 Torna seu site mais acessível para pessoas que usam leitores de tela — afinal, todos merecem uma boa experiência online!
🔧 Facilita a vida dos desenvolvedores, que conseguem ler e manter o código com mais facilidade.
